# Bash Notes
- pwd - Print Working Directory
- cd - Change Directory
- mkdir - Make a Directory
- touch - Create a File
- ls - List Contents of a Directory
 -- -a - List All (including hidden)
        - -l - list in a verbose format
- rm - Remove/Delete a File/Directory
 -- -r - Remove Recursively
 -- -f - Force
- nano - Open the Nano Text Editor
- rmdir - Delete a Directory
- vim - Open the Vim Text Editor

- ~ - HOME directory
- Absolute Path - a file path starting with / that specifies a unique location
- Relative Path - a file path starting with . that defines the relation to the current working directory
 -- .. - used with Relative paths to denote the parent directory

## Git Notes

- git add <filename> - adds a file and its changes to staging
- git commit -m "<Message>" - create a saved state with a message
- git status - check the status of the changes in the current repo
- git log - see the commit history for the current repo
- git push - push commits to remote
